### API para estudos

[Nest](https://github.com/nestjs/nest) framework TypeScript.

### Instalation

```bash
$ npm install
```

### Initialize docker container

This step will initialize container and run NodeJS with command<br>
``npm run start:dev``

```bash
$ docker-compose up
```

### Access API

```
http://localhost:3000/api
```
